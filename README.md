# AI-Accelerated Personal Portfolio

A futuristic, cyberpunk-themed portfolio built with a focus on Efficiency, Robustness, and Design.

## 🚀 Features
- **Cyberpunk Aesthetic:** Glitch effects, scanlines, and high-contrast neon design.
- **Interactive UI:** Smooth scrolling, intersection observers for animations, and a seamless marquee for technical skills.
- **Multimedia Experience:** Background video integration in the skills section and a custom background music player with volume controls.
- **Bilingual Support:** Full toggle between English and Spanish.
- **Security Hardened:** Implemented Content Security Policy (CSP), email obfuscation, and protected external links (`noopener noreferrer`).

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+).
- **Icons:** Devicon, FontAwesome.
- **Fonts:** Google Fonts (Orbitron, Rajdhani).
